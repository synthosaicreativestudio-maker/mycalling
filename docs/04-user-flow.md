# User Flow

## Верхнеуровневый путь
Лендинг -> Начать диагностику -> Контакт -> Профиль ученика -> Диагностика -> Обработка -> Отчет -> Следующее действие

## Подробный сценарий
1. Пользователь открывает `synthosai.ru`.
2. Видит promise продукта и CTA «Пройти диагностику».
3. Оставляет имя и email/Telegram.
4. Указывает класс, город, интересы и учебный контекст.
5. Проходит диагностику по блокам.
6. Видит экран расчета результатов.
7. Получает отчет.
8. Переключается между основным отчетом и parent summary.
9. Выбирает следующее действие.
