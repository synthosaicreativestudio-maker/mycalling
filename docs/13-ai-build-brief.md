# AI Build Brief

Собрать web MVP проекта МоеПризвание.

## Цели
- создать аккуратный современный интерфейс;
- провести пользователя через единый семейный сценарий;
- собрать контакт в начале;
- показать серьезный, но понятный результат.

## Что обязательно должно работать
- landing;
- onboarding;
- диагностика;
- расчет профиля;
- карьерный отчет;
- parent summary;
- CTA после отчета.

## Ограничения
- не использовать black-box AI для core scoring;
- не усложнять архитектуру;
- web-first;
- код должен быть готов к расширению.
