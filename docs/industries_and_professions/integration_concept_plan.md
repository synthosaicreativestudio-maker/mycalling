# Концептуально-технический план интеграции отраслей и профессий в проект «МоёПризвание»

Этот документ содержит детальный план реализации интеграции 20 отраслей и 119 профессий на основе лучших мировых профориентационных практик (**YouScience**, **Sokanu CareerExplorer**, **Gallup CliftonStrengths** и **Stanford Life Design Lab**).

---

## 1. Теоретико-методологическая рамка

Интеграция опирается на четыре научно обоснованных вектора сопоставления 150+ характеристик цифрового профиля подростка с требованиями профессиональных ролей:

```
                  +-----------------------------------+
                  |      МАТРИЦА СОПОСТАВЛЕНИЯ        |
                  +-----------------------------------+
                                    |
      +-----------------------------+-----------------------------+
      |                             |                             |
[Способности (YouScience)]   [Действие (Gallup)]   [Адаптивность (Савикас)]   [Идентичность (Марсия)]
- Когнитивный слой          - Роли Белбина         - Забота о будущем         - Формат выдачи отчета
- Исполнительные функции     - Командная динамика   - Уверенность в силах      - Уровень готовности
```

1.  **Вектор Способностей (Aptitudes - YouScience):**
    *   *Назначение:* Преодоление искажения восприятия (*Exposure Bias*), когда подросток выбирает только модные профессии.
    *   *Связь с базой:* Когнитивные шкалы (`cog_math_logic`, `cog_spatial`, `cog_verbal`) и исполнительные функции (`exec_inhibition`, `exec_flexibility`) сопоставляются с эталонными порогами сложности профессий.
2.  **Вектор Действия (Execution - Gallup CliftonStrengths):**
    *   *Назначение:* Определение роли подростка внутри профессиональной среды.
    *   *Связь с базой:* Роли Белбина (`belbin_leader`, `belbin_doer`, `belbin_creator`, `belbin_peacemaker`) определяют поведенческий профиль рекомендаций.
3.  **Вектор Адаптивности (Adaptability - Савикас):**
    *   *Назначение:* Оценка готовности к освоению новых сфер и преодолению трудностей.
    *   *Связь с базой:* Шкалы адаптивности (`savickas_concern`, `savickas_control`, `savickas_curiosity`, `savickas_confidence`) регулируют сложность предлагаемых первых шагов.
4.  **Вектор Идентичности (Identity - Марсия):**
    *   *Назначение:* Адаптация структуры отчета под уровень зрелости подростка.
    *   *Связь с базой:* При статусе «Мораторий» (`marcia_moratorium`) дается широкий спектр альтернатив; при «Достижении» (`marcia_achievement`) — конкретный план действий.

## 2. Архитектура интеграции по этапам сервиса

### Этап 1. Чат-сессия с коучем Романом (Мягкие пробы в реальном времени)

*   **Механика:** ИИ-коуч Роман использует проективные вопросы для выявления склонностей и предлагает подростку гипотетические сценарии (проекции).
*   **Синтаксис ИИ:** ИИ оборачивает названия профессий в обратные кавычки (например, `` `Гейм-дизайнер` ``).
*   **Фронтенд-компонент `InteractiveJobBadge`:**
    *   Парсит сообщения в реальном времени.
    *   Рендерит названия в виде анимированных неоновых кнопок с микро-свечением.
    *   При клике открывает **Slide-over (боковую) панель** с краткой выжимкой:
        *   Суть профессии (понятным языком).
        *   Ключевые предметы ОГЭ/ЕГЭ.
        *   Связь с его талантами («Почему Роман это предложил»).

### Этап 2. Алгоритм многомерного скоринга (Бэкенд-расчет)

При завершении диагностики API-эндпоинт `/api/v1/diagnostic/next-question` запускает расчет мэтчинга:

*   **Входные данные:** `SummaryProfile` (записанный в БД `DigitalProfile.summary`).
*   **Алгоритм:**
    1.  Для каждой профессии из `professionsDb` рассчитывается расстояние Хэмминга или коэффициент корреляции Пирсона по осям RIASEC, Gardner и Big Five.
    2.  Рассчитывается модификатор среды (совпадение предпочтений по удаленке/офису, командности).
    3.  Вычисляется итоговый `% соответствия` (Matching Score).
*   **Выходные данные:** Список топ-10 профессий с разбивкой по отраслям сохраняется в `prisma.diagnosticResult` в поле `scores`.

### Этап 3. Личный кабинет (Интерактивная визуализация)

Для создания Wow-эффекта и снижения стресса выбора внедряются три визуальных модуля:

#### А. Интерактивная 3D-карта «Созвездия Отраслей»
*   Реализуется на базе библиотеки Three.js или SVG-интерактивного холста с поддержкой жестов/зума.
*   Каждая из 20 отраслей — это светящаяся звездная сфера. Интенсивность свечения зависит от мэтчинга подростка с этой отраслью.
*   При клике сфера плавно масштабируется, показывая звезды-профессии внутри нее.

#### Б. Odyssey Plans («Проекты Одиссеи» по Stanford DYL)
Интерактивный таймлайн для трех параллельных траекторий развития:
1.  **Траектория А (Фокус):** Наиболее очевидный путь (прямой мэтч способностей и интересов).
2.  **Траектория Б (Гибрид):** Смежный путь на стыке талантов (например, IT в медицине).
3.  **Траектория В (Вызов):** Путь по сильным сторонам при слабом текущем интересе (развитие скрытого потенциала).

#### В. Трекер микро-проб (Life Design Roadmap)
Пошаговый чеклист задач для практического «прототипирования» выбранной профессии прямо из ЛК:
*   *Послушать подкаст с экспертом* (+10 к адаптивности).
*   *Решить кейс-песочницу* (+20 к уверенности).
*   *Изучить требования вузов* (+5 к заботе о будущем).
