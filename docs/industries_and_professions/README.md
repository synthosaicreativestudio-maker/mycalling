# Методология связи отраслей, психологических тестов и профессий

Этот документ описывает научную и методологическую основу базы данных профессий проекта «МоёПризвание».

---

## 1. Концепция триангуляции (Хочу — Могу — Как)

Рекомендация профессий строится на пересечении трех сигнальных слоев личности подростка:

```
          [ ХОЧУ ] (Интересы: RIASEC)
             /   \
            /     \
   [ МОГУ ] ------- [ КАК ] (Характер: Big Five)
(Интеллект: Gardner)
```

1. **Хочу (Интересы):** Оценивается по 6 шкалам **RIASEC** (Holland Codes):
   - **Realistic (R):** Реалистичный (техника, инструменты, природа, спорт).
   - **Investigative (I):** Исследовательский (аналитика, наука, данные, логика).
   - **Artistic (A):** Артистичный (творчество, дизайн, самовыражение).
   - **Social (S):** Социальный (помощь людям, обучение, общение).
   - **Enterprising (E):** Предпринимательский (лидерство, продажи, проекты, риски).
   - **Conventional (C):** Конвенциональный (структура, правила, таблицы, законы).
2. **Могу (Способности и Интеллект):** Оценивается по 8 шкалам множественного интеллекта Говарда **Гарднера**:
   - **Linguistic:** Лингвистический (работа со словом, тексты).
   - **Logical-Mathematical:** Логико-математический (анализ, расчеты, программирование).
   - **Spatial-Visual:** Пространственно-визуальный (дизайн, 3D, архитектура).
   - **Bodily-Kinesthetic:** Телесно-кинестетический (моторика, спорт, хирургия).
   - **Musical:** Музыкальный (чувство ритма, звука).
   - **Interpersonal:** Межличностный (эмпатия, переговоры, лидерство).
   - **Intrapersonal:** Внутриличностный (рефлексия, самоконтроль).
   - **Naturalist:** Натуралистический (живые системы, экология, биология).
3. **Как (Личностные трейты):** Оценивается по шкалам **Big Five (OCEAN)**:
   - **Openness (O):** Открытость новому (креативность, любознательность).
   - **Conscientiousness (C):** Добросовестность (организованность, дисциплина).
   - **Extraversion (E):** Экстраверсия (общительность, энергия).
   - **Agreeableness (A):** Доброжелательность (кооперация, эмпатия).
   - **Stability (N):** Эмоциональная стабильность (стрессоустойчивость).

## 2. Структура разметки профессий

Каждая профессия в файле [professions_db.ts](file:///Users/verakoroleva/Desktop/идеи/Мое Призвание/app/data/professions_db.ts) имеет жесткую схему:
- `riasec`: Массив ведущих типов (1-3 буквы в порядке значимости).
- `gardner`: Ведущие каналы интеллекта.
- `bigFive.traits`: Предпочтительный уровень черт (например, `high` для экстраверсии у PR-менеджера).
- `subjects`: Перечень школьных предметов РФ для сдачи на ЕГЭ/ОГЭ (позволяет сформировать образовательную траекторию).
- `skillFormula`: Формула из 3 мета-компетенций (skills-based).
- `transferableTo`: Список смежных направлений для обеспечения гибкости выбора.

## 3. Вычисление индекса согласованности (Consistency Index)

Индекс согласованности сравнивает качественные данные из чата с коучем Романом и результаты тестов:
- Если ученик в диалоге говорит: *«Я обожаю общаться с людьми, хочу вести переговоры»*, но в тесте Big Five показатель **Extraversion** равен `15%`, система снижает индекс согласованности и генерирует специальный раздел в отчете: **«Внутренние противоречия — твои скрытые ресурсы»**.
- Если расхождений нет, отчет помечается бейджем **«Высокая достоверность профиля»**.
