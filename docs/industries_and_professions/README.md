# Методология связи отраслей, психологических тестов и профессий

Этот документ описывает научную и методологическую основу базы данных профессий проекта «МоёПризвание».

---

## 1. Концепция триангуляции и 7-кластерная модель

Рекомендация профессий строится на основе глубокого цифрового профиля подростка, включающего 120 параметров, распределенных по 7 ключевым кластерам. Эти кластеры позволяют реализовать классическую схему «Хочу — Могу — Как» с учетом специфики подросткового возраста:

```
          [ ХОЧУ ] (Интересы: RIASEC, подшкалы CABIN, предметы)
             /   \
            /     \
    [ МОГУ ] ------- [ КАК / КОНТЕКСТ ] (Личность, Адаптивность, Семья)
(Интеллект: ICAR,
Исполнительные функции)
```

### 7 Кластеров Цифрового Профиля:
1. **Идентичность и Самовосприятие (identity):** VIA-силы характера, уровни академической/социальной самооценки, статусы профессиональной идентичности по Марсии (диффузия, мораторий, достижение).
2. **Когнитивный и Обучающий профиль (cognitive):** Общий интеллект, шкалы ICAR (вербальный, логический, пространственный), исполнительные функции (когнитивная гибкость, тормозной контроль), стили обучения (глубокий/поверхностный), метакогниции.
3. **Интересы и Склонности (interests):** 6 базовых типов RIASEC, узкие интересы CABIN (визуальное/сценическое искусство), цифровые хобби и предметные тяги.
4. **Личность и Эмоции (personality):** Большая Пятерка (BFI-10), эмоциональный интеллект (TEIQue), волевые черты (Grit, проактивность) и стресс-реактивность.
5. **Социальное взаимодействие (social):** Командные роли Белбина для подростков, ассертивность, качество связей, буллинг-резистентность и влияние семьи/сверстников.
6. **Поведенческие паттерны и Адаптивность (behavior):** Карьерная адаптивность по Савикасу (Забота, Контроль, Любознательность, Уверенность), барьеры (прокрастинация, перфекционизм, страх неудачи), гибридное сотрудничество Человек+ИИ.
7. **Контекст и Ограничения (context):** Географическая мобильность, финансовое состояние семьи, успеваемость, ограничения по здоровью.

## 2. Структура разметки профессий

Каждая профессия в файле [professions_db.ts](file:///Users/verakoroleva/Desktop/идеи/Мое Призвание/app/data/professions_db.ts) имеет жесткую схему:
- `riasec`: Массив ведущих типов (1-3 буквы в порядке значимости).
- `gardner`: Ведущие каналы интеллекта.
- `bigFive.traits`: Предпочтительный уровень черт (например, `high` для экстраверсии у PR-менеджера).
- `subjects`: Перечень школьных предметов РФ для сдачи на ЕГЭ/ОГЭ (позволяет сформировать образовательную траекторию).
- `skillFormula`: Формула из 3 мета-компетенций (skills-based).
- `transferableTo`: Список смежных направлений для обеспечения гибкости выбора.

## 3. Вычисление индекса согласованности (Consistency Index)

Индекс согласованности сравнивает качественные данные из чата с коучем Романом и результаты тестов:
- Если ученик в диалоге говорит: *«Я обожаю общаться с людьми, хочу вести переговоры»*, но в тесте Big Five показатель **Extraversion** равен `15%`, система снижает индекс согласованности и генерирует специальный раздел в отчете: **«Внутренние противоречия — твои скрытые ресурсы»**.
- Если расхождений нет, отчет помечается бейджем **«Высокая достоверность профиля»**.
