3. industry — точное совпадение со строкой из списка ОТРАСЛИ.
4. tier: массовая с большим спросом → everyday; растущая/технологичная → future;

## Как пользоваться

1. Скопируй блок выше, вместо `<ВСТАВЬ СЮДА…>` вставь партию названий (20–50 за раз — не перегружай).
2. Получишь JSON-массив паспортов.
3. Проверь глазами RIASEC-коды у 3–5 записей (главный ручной контроль).
4. Преобразуй JSON в TS-объекты и вставь в `app/data/professions_db.ts`
   (или дай нашему коду — я добавлю режим импорта JSON в `professions_db`).
5. Прогони `npx vitest run app/data/professions_db.test.ts` — валидатор поймает
   битые коды, несуществующие отрасли, дубли id/name.
