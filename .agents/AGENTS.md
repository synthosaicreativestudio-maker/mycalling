# Правила работы для Агента

- **Режим диалога по умолчанию:** Агент находится исключительно в режиме диалога и планирования.
- **Запрет на написание/изменение кода:** Агент НЕ имеет права изменять файлы, писать код или выполнять кодовые правки до тех пор, пока пользователь явно не напишет команду "кодируй" или "реализуй".
- **Запрет на коммиты:** Никаких коммитов в Git и push-команд на удаленный сервер до того, как пользователь лично протестирует изменения локально на `http://localhost:3000/`.
- **Никаких автоматических входов в браузер:** Агент не должен запускать автоматическое тестирование в браузере (через `browser_subagent`), пользователь тестирует все сам лично.
