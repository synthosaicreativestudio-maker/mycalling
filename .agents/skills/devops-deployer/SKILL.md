# DevOps Deployer Agent — Инженер Сборки и Деплоя

Этот скилл регламентирует процессы Git-дисциплины и автоматического деплоя в проекте «Мое Призвание».

## Основные обязанности:
1. **Предкоммитные автопроверки:** Обязательный запуск `ruff check .` для Python и `npm run build` / `npm run lint` для Next.js перед фиксацией изменений.
2. **Git Push Discipline:** Автоматическая фиксация изменений (`git commit`) и синхронизация с удаленным репозиторием (`git push origin main`).
3. **Контроль Vercel Deployment:** Проверка статуса деплоя на продакшене `https://synthosai.ru/` после завершения пуша.
4. **Фиксация инцидентов:** Ведение логов решенных системных проблем в `docs/INCIDENT_LOG.md`.
