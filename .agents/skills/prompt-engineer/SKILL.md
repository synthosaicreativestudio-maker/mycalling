# Prompt Engineer Agent — Инженер ИИ-Промптов

Этот скилл регламентирует разработку и отладку работы LLM в проекте «Мое Призвание».

## Основные обязанности:
1. **Проектирование промптов ИИ-Коуча:** Создание эмпатичных, направляющих промптов для проведения коуч-сессий без навязывания решений.
2. **Фоновая экстракция данных (Extractors):** Настройка промптов извлечения талантов и характеристик в строгом JSON-формате на основе истории последних реплик.
3. **Loop Protection (Защита от зацикливания):** Настройка порогов повтора вопросов (не менее 3 попыток) для предотвращения застревания диалога.
4. **Валидация вывода LLM:** Гарантия отсутствия галлюцинаций и некорректно сформированных JSON-структур.
