# Скилл: ИИ и Инжиниринг дизайна (AI Design & Engineering)

Этот скилл регулирует использование современных ИИ-инструментов для проектирования визуальных интерфейсов, генерации высококачественных графических ассетов и обеспечения технически правильной реализации дизайна в коде.

## 1. Стек ИИ-инструментов (AI Design Stack)
*   **Генерация графики и медиа (AI Image Gen):**
    *   *Midjourney / Flux / Ideogram / Recraft:* Для создания уникальных изображений, фонов, концептуальных 3D-моделей и иллюстраций.
    *   *Стиль генераций:* Поддерживайте визуальную консистентность (единая цветовая гамма, фокусное расстояние, уровень детализации).
*   **ИИ в продуктовом дизайне (AI UI/UX & Automation):**
    *   *Claude / ChatGPT / Gemini / Perplexity:* Для написания микрокопирайтинга, структуры опросников, промптов для отчетов и продуктовых гипотез.
    *   *v0 / Galileo / Relume / Lovable / Bolt:* Для быстрого создания UI-каркасов, сеток и прототипов страниц.

## 2. Design Engineering (Связующее звено)
*   **Интеграция Дизайна и Кода:**
    *   *Семантическая верстка:* Верстка интерфейса должна строго соответствовать структуре макета и семантическим тегам HTML5 (использование `<main>`, `<section>`, `<article>`, `<button>`).
    *   *Tailwind CSS:* Использование переменных и кастомных утилит Tailwind в конфигурации проекта, повторяющих переменные (Variables) Figma.
*   **Дизайн-токены (Design Tokens):**
    *   Все цвета, радиусы, шрифтовые размеры и тени должны экспортироваться в виде переменных (например, CSS Custom Properties или JSON-словарей), что позволяет быстро менять темы оформления (Dark/Light mode).

## 3. Figma-to-Code Workflow
*   **Figma Dev Mode:** Использование инструментов разработчика в Figma для инспектирования отступов, радиусов, Variables и CSS-кода.
*   **Адаптивное мышление при разработке:**
    *   Реализация макетов с расчетом на различные брейкпоинты (`sm`, `md`, `lg`, `xl`).
    *   Использование Flexbox и CSS Grid вместо абсолютного позиционирования для гибкости интерфейса.
*   **GitHub Workflow для дизайна:**
    *   Коммит изменений стилей, шрифтов и медиа-ассетов с четкими коммит-сообщениями.
    *   Валидация визуальной сборки на продакшне после каждого деплоя.
